Q: What is NLPAssist+?
A: NLPAssist+ is a system designed to help students and faculty by answering natural language questions about university courses, documents, and FAQs.

Q: How does NLPAssist+ work?
A: It uses a Retrieval-Augmented Generation (RAG) approach. It retrieves relevant information from a document collection and uses a Large Language Model (LLM) to generate an answer.

Q: What are the components of NLPAssist+?
A: It consists of a backend NLP service (Python) handling document ingestion, embeddings, and retrieval, and a frontend application (Web or Mobile) for user interaction.

A: React, Next.js, or Laravel for web; React Native for mobile.

Q: What should I use for vector storage?
A: You can use FAISS for local development or detailed vector databases like ChromaDB or Qdrant.

A: Yes, you must not expose API keys in the frontend and implement basic request validation.
